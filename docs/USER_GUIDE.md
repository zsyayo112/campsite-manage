## 客户端页面（公开访问）

这些页面面向客户，无需登录即可访问。

### 1. 关于我们页面
**访问地址**: `http://118.195.179.34/about`

**功能说明**:
- 展示营地介绍、特色、位置等信息
- 显示营地联系方式
- 提供快捷导航到活动、套餐、预约页面

---

### 2. 活动列表页面
**访问地址**: `http://118.195.179.34/activities`

**功能说明**:
- 展示所有可预约的活动项目
- 支持按季节筛选（全部/冬季/夏季）
- 显示活动封面图、名称、价格、时长等信息
- 点击活动卡片可查看详情

**使用方法**:
1. 打开页面后默认显示所有活动
2. 点击顶部"冬季项目"或"夏季项目"按钮筛选
3. 点击任意活动卡片进入详情页

---

### 3. 活动详情页面
**访问地址**: `http://118.195.179.34/activities/:id`

**功能说明**:
- 展示活动的详细信息
- 图片轮播（支持左右切换）
- 显示活动亮点、详细介绍、注意事项
- 提供预约入口

**使用方法**:
1. 左右滑动或点击箭头切换图片
2. 查看活动详情和注意事项
3. 点击"立即预约"按钮进入预约页面

---

### 4. 套餐列表页面
**访问地址**: `http://118.195.179.34/packages`

**功能说明**:
- 展示所有精选套餐
- 显示套餐价格、原价、折扣信息
- 显示套餐亮点标签

**使用方法**:
1. 浏览所有套餐
2. 点击套餐卡片查看详情
3. 点击"立即预约"进入预约流程

---

### 5. 套餐详情页面
**访问地址**: `http://118.195.179.34/packages/:id`

**功能说明**:
- 展示套餐完整信息
- 显示包含的活动项目
- 显示套餐亮点和注意事项
- 提供预约入口

---

### 6. 在线预约页面
**访问地址**: `http://118.195.179.34/book`

**功能说明**:
- 填写预约信息（姓名、电话、人数、日期等）
- 选择要预约的项目或套餐
- 提交预约申请

**使用方法**:
1. 填写联系人姓名和电话（必填）
2. 选择预约日期和人数
3. 选择要预约的活动或套餐
4. 填写备注信息（可选）
5. 点击"提交预约"

---

### 7. 订单查询页面
**访问地址**: `http://118.195.179.34/order-query`

**功能说明**:
- 通过手机号查询预约订单
- 查看订单状态和详情

**使用方法**:
1. 输入预约时使用的手机号
2. 点击"查询"按钮
3. 查看订单列表和状态

## 管理后台（需登录）

管理后台需要账号密码登录，用于营地运营管理。

### 登录页面
**访问地址**: `http://118.195.179.34/login`

**默认账号**:
- 用户名: `admin`
- 密码: `admin123`（请登录后修改）

---

### 1. 仪表盘
**访问路径**: 登录后首页 或 侧边栏"仪表盘"

**功能说明**:
- 显示今日预约数、待处理订单数等统计
- 快速查看最新预约

---

### 2. 项目管理
**访问路径**: 侧边栏 → "项目管理"

**功能说明**:
- 管理所有活动项目
- 添加、编辑、删除项目
- 设置项目启用/停用状态
- 上传项目图片和视频

**使用方法**:

#### 新建项目:
1. 点击"新建项目"按钮
2. 填写基本信息（名称、描述、价格、季节等）
3. 切换到"图片视频"标签上传封面图和其他图片
4. 切换到"详细内容"标签填写亮点和注意事项
5. 点击"创建"保存

#### 编辑项目:
1. 在项目卡片上点击"编辑"按钮
2. 修改需要更改的信息
3. 点击"保存"

#### 启用/停用项目:
1. 点击项目卡片上的开关按钮
2. 停用后项目不会在客户端显示

---

### 3. 套餐管理
**访问路径**: 侧边栏 → "套餐管理"

**功能说明**:
- 管理所有套餐
- 添加、编辑、删除套餐
- 选择套餐包含的活动项目
- 设置套餐价格和折扣

**使用方法**:

#### 新建套餐:
1. 点击"新建套餐"按钮
2. 填写套餐名称、描述、价格
3. 选择包含的活动项目
4. 上传封面图
5. 点击"创建"

#### 编辑套餐:
1. 点击套餐卡片上的"编辑"按钮
2. 修改信息后点击"保存"

### 4. 预约管理
**访问路径**: 侧边栏 → "预约管理"

**功能说明**:
- 查看所有预约订单
- 处理预约（确认/取消）

#### 筛选订单:
1. 使用顶部筛选器按状态筛选
2. 使用搜索框搜索客户姓名或手机号

## 常见操作流程

### 流程1: 客户预约流程
```
客户访问网站 → 浏览活动/套餐 → 选择项目 → 填写预约信息 → 提交预约
                                                          ↓
管理员收到预约 ← 系统生成订单 ← 预约成功页面显示订单号
      ↓
确认/取消预约 → 通知客户
```

### 流程2: 添加新活动流程
```
登录管理后台 → 项目管理 → 新建项目 → 填写信息 → 上传图片 → 创建
                                                          ↓
客户端自动显示 ← 项目状态设为"启用"
```

### 流程3: 创建套餐流程
```
登录管理后台 → 套餐管理 → 新建套餐 → 填写信息 → 选择活动 → 创建
                                                          ↓
客户端套餐页面显示 ← 套餐状态设为"启用"
```

### 流程4: 处理预约流程
```
登录管理后台 → 预约管理 → 查看待处理订单 → 确认/取消 → 完成
```

## 移动端使用

所有页面均支持手机访问，移动端特性：

1. **导航菜单**: 点击右上角汉堡菜单图标展开导航
2. **图片轮播**: 支持左右滑动切换图片
3. **表单填写**: 输入框自动适配手机键盘
4. **底部弹窗**: 管理后台的表单在手机上从底部弹出

## 常见问题

### Q: 忘记管理员密码怎么办？
A: 联系技术支持重置密码，或通过数据库直接修改。

### Q: 图片上传失败怎么办？
A: 检查图片大小是否超过10MB，支持的格式为 jpg、png、gif、webp。

### Q: 客户看不到新添加的项目？
A: 检查项目状态是否已设为"启用"。

### Q: 如何修改联系电话？
A: 登录管理后台 → 设置 → 营地信息 → 修改联系方式。
